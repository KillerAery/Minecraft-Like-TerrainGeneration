# 介绍

---

DEMO简介：使用UE4和C++开发的Minecraft类游戏地形生成Demo

UE4版本：4.22

可运行平台：Windows 10

原理博客：[博客园 | Minecraft类游戏地形生成机制](https://www.cnblogs.com/KillerAery/p/11509470.html)

进度：正在美化代码加注释中....

# 功能

---

- 地形高度
- 雨水侵蚀、河流（未完）
- 生物群落
- 洞穴生成
- 植被生成
- 建筑生成
